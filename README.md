# 👶 Baby Naming Ceremony Invitation Website

A modern, bilingual (English & Kannada) invitation website created for our Baby Naming & Cradle Ceremony.

The website is fully responsive, installable as a Progressive Web App (PWA), and deploys automatically using GitHub Pages.

---

## ✨ Features

- 👶 Beautiful landing page
- 🌐 English & Kannada support
- 📜 Digital invitation card
- ⏳ Live countdown timer
- 📍 Google Maps navigation
- 📱 Mobile responsive design
- 🌙 Dark / Light mode
- 🎵 Background music support
- 🌸 Floating flower animation
- 🎉 Confetti celebration
- ❤️ RSVP form
- 📷 Photo gallery
- 📤 WhatsApp sharing
- 📅 Add to Google Calendar
- 📲 Progressive Web App (PWA)
- 📦 Offline support using Service Worker
- 🔍 SEO optimized
- 🚀 Automatic deployment with GitHub Actions

## 🚀 Deployment

1. Create a GitHub repository.
2. Push all project files.
3. Open **Settings → Pages**.
4. Enable **GitHub Pages**.
5. Your invitation will be available at:

```
https://YOUR_GITHUB_USERNAME.github.io/baby-naming-invitation/
```

---

## ⚙️ Customization

Before publishing, update the following:

- Replace `assets/invitation.jpg`
- Replace `assets/bg.jpg`
- Add `assets/music.mp3` (optional)
- Update the event date in `script.js`
- Update the Google Maps link
- Replace the contact number
- Add your gallery images
- Replace the placeholder icons
